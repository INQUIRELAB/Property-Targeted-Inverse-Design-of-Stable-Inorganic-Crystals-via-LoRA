# Property-Targeted-Inverse-Design-of-Stable-Inorganic-Crystals-via-LoRA
Official code for the property-targeted inverse design of stable inorganic crystals. By applying Low-Rank Adaptation (LoRA) to a GemNet-T diffusion model, this pipeline enables materials discovery on consumer GPUs. It delivers a 19x parameter reduction and 89% generative fidelity , fully validated by VASP DFT.
